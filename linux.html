Linux CyberPatriot Image Hardening

Security hardening guide for Linux systems

This guide provides a step-by-step approach to hardening a Linux CyberPatriot image for security using Bash scripts. The main.sh script gets to 100 points on tr2 ubuntu.

Overview

The Linux security hardening guide includes comprehensive steps for securing Ubuntu and other Linux systems used in CyberPatriot competitions.
